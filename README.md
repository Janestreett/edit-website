## 🛠️ Pembaruan pada Aktivitas 3
Sesuai dengan instruksi tantangan pada lembar kerja, saya telah menambahkan dan memodifikasi beberapa komponen web berikut:
1. **Kustomisasi Warna Teks (CSS Variables):** Menambahkan variasi warna aksen baru seperti hijau tren pasar (`--accent-green`) dan biru (`--accent-blue`) untuk memberikan penekanan visual yang lebih dinamis pada teks portofolio tanpa merusak estetika *dark mode* yang elegan.
2. **Implementasi Gambar Favorit:** Mengganti elemen inisial teks statis pada *frame avatar* dengan tag gambar (`<img>`) yang menampilkan grafik visual representatif pasar modal menggunakan teknik filter *grayscale* dan kontras agar menyatu dengan tema dasar.
3. **Penyematan Tautan Sosial Media:** Menambahkan baris tautan interaktif baru yang terhubung langsung ke akun sosial media eksternal aktif.
